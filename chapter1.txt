    1. js나 css를 단순히 브라우저로 열면 코드만 보이고 작동하는 부분이 없음. 
    -> 단순히 켜볼 생각을 안해봐서 처음 봄. 
    -> html이 접착제라고 설명 중. -> 나는 건물의 뼈대라고 생각한다. 

    2. css, js 엔진 두 개가 돌아가고 있다. 

    3. <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    - http-equiv : http 해더에 정보를 제공하는 것. 
    ex) content-security-policy : 문서에 대한 콘텐츠 보안정책을 명시한다. 
        content-type : 문서의 character 인코딩을 명시한다. 
        default-style : 사용할 스타일 시트를 명시한다. 

    - x-ua-compatible : 크로스 브라우징 문제로 IE(인터넷 익스프로러)를 사용할 때는 
    가장 최신의 표준문서를 사용하라는 내용을 명시하고 있다. 
    
    - compatible : 호환
